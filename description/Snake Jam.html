Event Description

Snake Jam is a fun event. 
It's much like snakes and ladders with a twist. Good actions allow the player to climb a ladder and get closer to finish, while bad decisions make the player slide down chute, which takes him farther from finish. 
So get ready to hop on the giant snake-ladder flex with the biggest dice ever.

 

Rules

  • Team size must be of 2 students.
  • A 10*10 game board on the floor is used to play the game.
  • Roll the dice. If an odd number is rolled, it is a free move, no questions asked. And if, an even number comes then a question is asked.
  • If question is answered correctly, move is made as per the number of the dice. No consequences of the wrong answered question.
  • If ladder is encountered, additional question is asked. If the question is answered correctly, player moves up the ladder. No consequences of the wrong answered question.
  • If snake is encountered, again, additional question is asked. If the question is answered correctly, player stays at his/her position. If the question is answered wrong, player has to reach the tail of the snake.
  • As many teams can play !
